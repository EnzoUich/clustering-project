# Clustering Project

## Usage

> Instructions are for Linux

### Put data in 'data' folder named 'data.csv'

```bash
mkdir data
touch data/data.csv
```

### Optional: Create virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install required libraries

```bash
pip install -r requirements.txt
```

### Run main.py

```bash
python3 main.py
```
